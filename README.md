# A discord incremental counter bot.
Uses the py-cord framework. Some additional goodies in `cogs/fun.py` are also implemented.

# Requirements
Python 3.12

# Installation
Run either:
```sh
pip install -r requirements.txt
```
or if Pipenv is installed:
```sh
pipenv sync
```

# Usage
Run the bot with
```sh
python bot.py
```
